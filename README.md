1. **사이버 법제도 개정**: 간첩법 개정으로 사이버 공간 간첩 처벌이 제외된 점, 국가정보원 개편으로 보안 정보체계 변화와 대응 전략 필요성  
2. **보안 인재 양성 체계**: BoB 프로그램 등 국가 차세대 보안인재 육성 시스템의 구조 및 평가 방식  
3. **AI+DB 융합**: AI 에이전트 시대에 데이터베이스의 역할 변화 및 빅데이터 처리 방법론
