## 예상 출제 포인트

1. **4세대 DBMS**: 방대한 실시간 데이터 처리를 위한 아키텍처 및 고가용성 설계, 재난 대비 데이터센터 보호 인프라(방공호 수준) 적용 사례
2. **동형암호**: 암호화된 상태에서의 연산 원리와 성능 한계, 공공·금융권 데이터베이스 보안 OS로서의 적용과 동형 컴퓨팅 개념
3. **통합 데이터베이스**: 개인정보 집중 관리 위험성과 효용, 마스터 데이터베이스 설계 원칙 및 개인정보보호법·정보통신망법 적용
4. **NIS2**: EU 지침의 국내 적용 방안, 기업별 보안 의무 강화 요건, CCTV·망분리 등 보안 인프라 교체 주기와 비용 산정
5. **사이버보안 지침**: 정보보안에서 사이버보안으로의 개념 확장, 정보통신망 전반에 대한 위험 평가 및 통제(N2SF) 반영 방안
