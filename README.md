## 예상 출제 포인트

1. **통합 데이터 아키텍처**: 데이터 레이크와 데이터베이스 통합의 필요성, 레이크하우스/레이크베이스 아키텍처의 개념 및 장단점, 현업에서의 적용 사례.
2. **AI 생성 데이터 신뢰성**: 의료 등 전문 분야에서의 AI 생성 딥페이크 위험, 데이터 품질 관리 및 검증 기술, 윤리적·기술적 대응 방안.
3. **AI 기반 자동 침투 테스트**: offensive security 관점에서 AI 활용, 공격 시뮬레이션 자동화의 기술적 메커니즘 및 한계.
4. **개방형 SIEM 및 에이전틱 보안**: 기존 SIEM의 한계, AI 에이전트 기반 위협 탐지/대응 아키텍처, 오픈소스/개방형 플랫폼의 중요성.
5. **OT/ICS 보안 및 글로벌 규제**: 운영기술 환경의 특성, 주요 글로벌 규제(예: CRA, NIS2) 내용 및 준비 전략, IT-OT Convergence 보안.
